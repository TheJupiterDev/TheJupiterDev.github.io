My new project: Dicey

November 21, 2024


In this post, I explain the plan for a new full stack web app: Dicey.


A web application that lets users create, play, and share custom board games. Think of it as a digital sandbox for creating tabletop experiences. Users can design the game board, upload custom pieces, set up rules, and play with friends in real-time or asynchronously.

The main point for this app is because I want more experience using frameworks like DJango, and I thought it would be a fun, new experience for me to make a web app with a Multiplayer support. Hopefully I will have a brand new blog post out once I get quite a bit more work done, but until then,
